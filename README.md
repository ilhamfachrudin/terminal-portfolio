# Terminal Portfolio

A retro-themed portfolio that requires users to type commands to navigate. Built to look like a CRT monitor terminal.

## Features
- Command-based navigation (`help`, `about`, `skills`, `projects`, `contact`, `social`)
- Command history (arrow keys)
- Cyberpunk neon styling
- Zero dependencies — pure HTML, CSS, JavaScript

## Commands
| Command | Description |
|---------|-------------|
| `help` | List all available commands |
| `about` | About me |
| `skills` | Technical skills |
| `projects` | My projects |
| `contact` | Contact information |
| `social` | Social media links |
| `clear` | Clear the terminal |
| `banner` | Show the ASCII banner |

## Usage
```bash
# Just open index.html in a browser — no build step needed
open index.html
```

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
